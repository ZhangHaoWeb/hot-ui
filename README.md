# hot-ui

> A Vue.js components.

<center>![](static/images/hot.jpeg)</center> 


## install

``` bash
npm install hot-ui -D
```

## Quick Start
```bash
	import Vue form 'vue'
	import hotUI from 'hot-ui'
	
	Vue.use(hotUI)
```

## Browser Support
Modern browsers and Internet <font color="orange">Explorer 10+</font>.
